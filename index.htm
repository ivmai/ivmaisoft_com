News JCGO

IvMaiSoft LLC

Founded in 2010, IvMaiSoft LLC is a privately held, software development and consulting company specialized mainly in providing advanced execution environments for Java applications.

Our flagship product

JCGO, a Java source to C code (java-to-c) translator, is the flagship software product developed, licensed and supported by IvMaiSoft LLC. With JCGO you can compile your Java application to machine native code (a binary executable file) making it run faster, consume less system resources, harder to decompile and easier to deploy. Even more, with JCGO you could also make your Java applications run on a wider range of operating systems, computer systems, embedded devices and programmable controllers.

JCGO is a fully Open Source project now.

For more details, please visit JCGO project page.

Core competency

IvMaiSoft LLC has the team of the qualified software developers those professional competency covers, at least, the following areas:

  • ahead-of-time (AOT) Java to C code (java2c) translation and to native machine executable (java2exe, java-to-native-code) compilation;
  • Java VM implementations (advanced in performance and code size, advanced in the level of portability, implementations for multiprocessor systems, implementations for small devices, implementations for programmable controllers with limited RAM size, etc.);
  • memory garbage collector (GC) implementations;
  • Java core class library implementations (optimized editions, integration with Java VMs, multi-vendor library merges);
  • Java graphics (GUI) back-ends;
  • serial ports communication in Java (rxtx);
  • ASN.1-based solutions.
